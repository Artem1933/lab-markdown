## 3. Діаграма Mermaid
Нижче наведена діаграма процесу:

```mermaid
graph TD;
    A[Ідея проєкту] --> B{Аналіз вимог};
    B -->|Вимоги чіткі| C[Розробка коду];
    B -->|Вимоги нечіткі| D[Уточнення у замовника];
    D --> B;
    C --> E[Тестування];
    E -->|Баги знайдено| C;
    E -->|Все працює| F[Реліз];

```
